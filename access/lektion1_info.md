# Lektion 1: Shortcuts & die erste eigene Webseite

**Ziele für 90 Minuten:**
1. Den Workflow verstehen: Datei herunterladen -> am richtigen Ort speichern -> in zwei Programmen gleichzeitig öffnen (Editor & Browser).
2. Den Unterschied zwischen Browser-Shortcuts und Screenreader-Shortcuts lernen.
3. Erste HTML-Tags (`<h1>`, `<h2>`, `<ul>`, `<li>`) anwenden.

**Rolle Begleitperson (EXTREM WICHTIG HEUTE):**
Die Schülerin muss heute den "Zauber-Kreislauf" lernen (Im Editor speichern, im Browser aktualisieren). 
- Helfen Sie ihr beim allerersten Schritt: Dem Speichern der Datei an einem Ort, den sie wiederfindet (z.B. Ordner "Informatik" auf dem Desktop).
- Helfen Sie ihr, die Datei mit dem Programm "Editor" (oder Notepad) zu öffnen. Ein normaler Klick öffnet sie oft nur im Browser!
- Lassen Sie die Schülerin beim HTML diktieren. Tippen Sie exakt das, was sie sagt.
- Helfen Sie bei Tastenkombinationen, indem Sie ansagen, wo sich Tasten wie `Ctrl` (Strg), `Alt` oder `F5` befinden.
